# Whack a Mole

Whack a Mole is a classic game with popular variations like gator panic that are still popular today, and this template makes it easy to create one (or many) of your own without having to learn a complicated framework. This template gives you a complete scaffold to start building right away.

## Visual Customization Controls (VCC's)
- 🎮 Customize game settings.
- 🖼️ Customize moles and backgrounds.
- ⁉ Customize mole reaction text.
- 🔈 Customize sounds and background music.
- 💅 Customize the colors and visual style.
- ⚙️ Add Google Analytics ID and Open Graph information for sharing.
